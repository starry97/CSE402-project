About MViz

Current machine learning visualization tools out there either are too tedious to use for simple models, like draw.io, or do not provide user enough controls, like the embedded visualization function in Keras. In this project, we fill this gap by presenting an easy-to-use and customizable visualization library, which parses existing Keras models and generates visualizations.

Instruction

load a Keras summary file or a JSON file produced by MViz.
